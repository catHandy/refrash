## 개요

냉장고와 찬장에 무엇이 있는지 실시간으로 파악하고, 식사 기록과 재료 소비를 연동해 잔량을 자동으로 관리하는 살림 도구입니다.

- **왼쪽 패널** — 찬장 / 냉동실 / 냉장실 단면 뷰. 아이템 클릭 시 수정·삭제 모달
- **오른쪽 패널** — 재료 추가 / 식사 기록(재료 자동 차감) / 30일 히스토리 아코디언

유통기한이 임박하거나 만료된 식재료는 자동으로 흑백 처리 + D-day 뱃지로 표시됩니다.

## 기술 스택

| 항목 | 내용 |
|------|------|
| 렌더링 | React 18 (Babel standalone, CDN) |
| 스타일 | Vanilla CSS (CSS variables 테마 시스템) |
| 타이포 | IBM Plex Sans KR + IBM Plex Mono |
| 데이터 | `localStorage` (브라우저 로컬 저장) |
| 번들러 | 없음 — 단일 HTML + 분리 JSX 파일 |

별도 빌드 없이 `냉장고 인벤토리.html`을 브라우저에서 열면 바로 실행됩니다.

---

## 로컬 실행

```bash
git clone https://github.com/your-username/your-repo.git
cd your-repo
# 아무 로컬 서버로 실행 (file:// 프로토콜은 Babel 로드 제한 있음)
npx serve .
# 또는
python3 -m http.server 8080
```

브라우저에서 `http://localhost:8080/냉장고 인벤토리.html` 접속

## 파일 구조

```
├── 냉장고 인벤토리.html   # 메인 엔트리 + 전체 CSS
├── data.js               # 데이터 모델, 샘플 시드, localStorage 헬퍼
├── fridge.jsx            # 냉장고 단면 컴포넌트 + 아이템 모달
├── panel.jsx             # 현황판 — 통계, 재료추가, 식사기록, 히스토리
├── app.jsx               # 루트 앱 — 상태 관리, 레이아웃, Tweaks 연결
└── tweaks-panel.jsx      # 인앱 디자인 트윅 패널 (테마/컬러/설정)
```
